Sobre o Projeto

Ferramenta Web com foco no aprendizado!

O Projeto HANOI é uma plataforma educacional que permite a criação de quizzes, integrando uma experiência gamificada. Nosso objetivo é resolver as limitações de ferramentas semelhantes, oferecendo uma solução gratuita, de fácil usabilidade e sem restrições quanto ao número de usuários ou atividades.
